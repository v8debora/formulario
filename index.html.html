Questionário:

a- Na tag "input" com atributo type="text", para que serve o atributo "placeholder"?

Resposta: O placeholder atributo especifica uma dica curta que descreve o valor esperado de um campo de entrada (por exemplo, um valor de amostra ou uma breve descrição do formato esperado).


b- Na tag "input" com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

Resposta: Use o atributo checked para indicar se o item está selecionado por padrão.


c- Na tag "input" para que serve o atributo "value"?

Resposta: Você deve usar o atributo value para definir o valor a ser enviado por este item.


d- Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

Resposta: Para usar este tipo de campo de um formulário, temos que fornecer o valor "radio" ao atributo type da tag "input".


e- Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?

Resposta: O elemento HTML "label" representa uma legenda para um item em uma interface de usuário. O atributo "for" serve para especificar em em qual caixa o texto ficará colado.


f- Na tag "option" para que serve o atributo "selected"?

Resposta: O atributo "selected" serve para criar uma lista de "drop-down" (caixa de escolha no formulario).


g- Nas tags de formulário para que serve o atributo "required"?

Resposta: atributo ""required" serve para especificar que o usuário deve preencher o campo com o atributo para enviar.


h- Na tag "form" para que serve o atributo "action"?

Resposta: O atributo especifica para onde enviar os dados do formulário quando um formulário é enviado.


i- Na tag "form" para que serve o atributo "method"?

Resposta: O atributo especifica como enviar dados de formulário, pode ser enviado com as variáveis "get" ou "post".


j- Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form"?

Resposta: "password".


k- Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos?

Resposta: Os dados do formulário podem ser enviados como variáveis de URL (com method="get") ou como transação de postagem HTTP (com method="post").
